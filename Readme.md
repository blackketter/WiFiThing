# WiFiThing library

Simple framework setting up WiFi services on Arduino ESP8266 

Includes:

- Connecting to wifi
- Setting a hostname
- Handling OTA updates
- Run NTP client (and drive system clock)
- Notifications when wifi goes up and down
- Simple HTTP getter
